## Description

[Nest](https://github.com/nestjs/nest) Starter project

## Installation


```bash
# install dependencies
$ npm install
```

## Running the app

```bash
# development
$ npm start

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
